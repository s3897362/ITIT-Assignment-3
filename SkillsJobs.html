Skills and Jobs

Programmer

Overview

As a programmer you will be responsible for coding the Synthesizer. Individuals should understand the appropriate functions and algorithms required to create connections with each component and the given code. Programmer will work with the hardware engineer and software tester to create these connections.

Skills Needed

  • Be knowledgeable in C# as well as Arduino UNO code, at least a year of experience in both languages.
  • Proficient in web programming such as HTML5/CSS3/Javascript
  • The job would need you to add additional code
  • Understand algorithms and functions which can be applied to the arduino uno
  • Problem solving skills

Responsibilities

  • Essential that they are hard working and proficient in C# and Arduino IDE.
  • Must be able to work well in a team based environment.
  • Understand how to create connections between code and hardware.

hardware engineer

Overview

As a programmer you will be responsible for developing the hardware for the synthesizer. Individuals will need to be able to identify how to plan and develop circuits which will appropriately communicate between each other. The Hardware engineer will work with the programmer and the software tester to create these connections.

Skills needed

  • Have engineering skills
  • Have complex mathematical skills required for engineering advanced computing components.
  • Vast knowledge in coding and hardware
  • Soldering Certificate
  • Able to efficiently create circuits

Responsibilities

  • Essential that they are hard working and understand how to create. circuits on an Arduino.
  • Must be able to work well in a team based environment.
  • Understand which resources are the most necessary, and how to create circuits using the resources available.

Software Tester

Overview

As a software tester you will be responsible for testing the code developed from the programmer. You will need to understand how to create test plans, and how to appropriately test the software. This testing will further need to identify possible logical errors and mistakes within the software, and mistakes will be corrected with the help of the programmer. The software tester will work with the Programmer and the Hardware Engineer.

Skills Needed

  • Be knowledgeable in C# as well as Arduino UNO code.
  • Understanding how to test C# and Arduino code.
  • Problem solving skills
  • Understand how to resolve coding issues
  • Critical thinking
  • Able to manually and automatically test code.
  • Create test plans

Responsibilities

  • Understand how to create test plans and appropriately test the software.
  • Apply manual or automatic tests to the synthesizers software.
  • Understand how to resolve issues if there are any within the code.
  • Must be able to work well in a team based environment.
