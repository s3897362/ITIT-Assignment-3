Overview

Topic

Overall this project is aiming to prove the possibility of creating a unique synthesiser sound that is special to a particular board. Ideally this could be done with any number of boards such as Arduinos, Raspberry Piâ€™s and many others, but the core concept is a compact, unique and functional little synthesiser using board based technology with a unique sound depending on the board used.

If we were to see this project through to the end we would be looking to see a fully functioning and polished synthesiser with an arduino or other board based brain running the show. In terms of the underlying code used for the synthesiser, the Arduino Uno utilises C# and therefore requires us to learn some fundamental concepts of the language. The code allows the components of the synthesiser to interact with one another.

Motivation

This is a musical passion project from Matthew that combines a love for instruments and small board based technologies. The project allows for the skills that we have learned and will continue to learn throughout our IT courses and combines them with a passion to create a unique product that is easily accessible and functional.

Landscape

An arduino based synthesiser is not a new concept, there have been many successful examples and has overall been a common idea hence why simply making an arduino synthesiser is not an innovative idea. In regards to the unique sound on each arduino there is a similar concept that has been used in synthesisers, though not identical and that is the DNA function on a synthesiser called the OP-1 made by teenage engineering, this consists of each device having a slightly different unique preset which is one of the many sounds the OP-1 can produce. The main point of difference of our proposed project and a synthesiser such as the OP-1 would very much be the scale of the endeavour, ours is a very small board of minimal cost running customisable code where as the OP-1 is using a premade preset that might differentiate from other units slightly and since it is professionally produced it has a price tag to match.
